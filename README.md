# RENAN TEIXEIRA — Signature Page

Página de identidade visual em tela única feita com Next.js, TypeScript, WebGL e Canvas 2D.

## Desenvolvimento local

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Build de produção

```bash
npm run build
npm start
```

## Deploy na Vercel

1. Envie esta pasta para um repositório GitHub.
2. Na Vercel, escolha **Add New > Project**.
3. Importe o repositório.
4. A Vercel detectará Next.js automaticamente.
5. Clique em **Deploy**.

Nenhuma variável de ambiente é necessária.
